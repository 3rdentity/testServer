Frequently Asked Questions


- Why?
Because I want to make it easier for people to host servers (particularly game servers), play with friends, and connect with others. I've been particularly frustrated with the state of server hosting, you either have to mess around with port-fowarding (difficult, many routers can't do it) or pay a company $10-$20 a month to host a server. With Grappl, you can put up a server with no hassle, running off your own machine, in a moment.
- How does it work?
It works by tunnelling data through Grappl relay servers. The exact details are complex, but Grappl isn't built with support for any specific protocol, and just works as a dumb TCP tunnel. That means any application that uses TCP and binds to only one port should work through it.
- Is it safe?
It's safe. Nothing nefarious is done with data sent through Grappl.
You can also view the source code of the client here
- What applications/games will it run with?
It will run on any game that has a TCP client which accepts alternate ports from the default.
This includes Minecraft, Starbound, Terraria, and most other games. It can even be used to host HTTP servers!
- How do you pay to keep this running?
Donations, and donations alone! Please donate if you like Grappl. You can alpha test new features if you do!

- How can I set up a MC server with Grappl?
This video does a great job of explaining how. The explanation starts at 1:10.

If you want to host a dedicated server, it's a bit more complicated.
You have to download the MC server application from minecraft.net, run it, then enter '25565' as the server port into Grappl.

- It won't work on my computer, what's wrong?
If you don't have Java installed, you need to have that installed.
If the application starts, you've let it through your firewall, and you get a public address (but you can't connect to it) any number of things could be wrong, but it's usually just the wrong port was entered.
If it's being a total pain, you can contact me on Twitter @Cactose and I'll look into helping you.
- Grappl is java? Is there an API?
You bet! The documentation is on the github page. It's super simple.
- Can I run my own relay servers?
Absolutely! Contact me at matt@daexsys.com and we can work out licensing. The serverside is closed source. Self-hosting a relay server can be preferable, for example, if you are a game dev and are using Grappl for your game's networking through the API. You are still free to use the public relay servers in that scenario, but it may not be preferable.
